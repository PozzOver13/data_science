    \section{Chapter 4. NumPy Basics: Arrays and Vectorized
Computation}\label{chapter-4.-numpy-basics-arrays-and-vectorized-computation}

NumPy, short for Numerical Python e' una libreria fondamentale in quanto
molte delle funzioni di base per il calcolo numerico vengono da
utilizzate da molte altre librerie

\begin{itemize}
\tightlist
\item
  Array
\item
  Funzioni di calcolo, Algebra lineare
\item
  API per C++
\end{itemize}

Il motivo principale del successo di numpy e' dovuto all'efficienza
delle sue operazioni. Spesso le funzioni sono ottimizzate per essere
vettoriali e non necessitano dei for loop. Inoltre a livello piu' basso
Numpy alloca un blocco di memoria per ogni oggetto a cui C puo' accedere
senza controlli e rallentamenti per via della sua API.

RK = in numpy le sotto selezioni di un array sono solo delle "viste",
questo significa che modificare le viste significa modificare anche
l'array di provenienza. Questo e' un potenziale side effect della
gestione efficiente della memoria di Numpy (sarebbe complesso copiare
ogni oggetto, soprattutto se di grandi dimensioni)

\begin{itemize}
\tightlist
\item
  axis 0 = rows
\item
  axis 1 = columns
\end{itemize}
